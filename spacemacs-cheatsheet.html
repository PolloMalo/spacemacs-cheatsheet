Spacemacs Cheatsheet

Modified February 18, 2021

Common

Exit
:wqa ENTER save and exit
:qa! exit without saving
Move
h left
l right
k up
j down
w next word
e end of word
$ end of line
^ start of line
gg end of buffer
G start of buffer
:{line number} go to line
spc w j window below
spc w k window above
Select
v visual select
V visual line select
Delete
x delete at cursor
[n] de delete to end of word
[n] dw delete to end of word and trailing space
[n] d$ delete to end of line
[n] dd delete line
[n] cw delete to end of word, then edit
[n] c$ delete to end of line, then edit
d delete visual selection
Copy (Yank)
yw copy word under cursor
yy copy line under cursor
y copy visual selection
Paste
p paste
:r{filename} paste contents from another file
Find
/{text} ENTER Find forward
?{text} ENTER Find backward
n repeat search
N repeat search in reverse direction
SPC s c clear highlighted search results
% find matching paren or bracket
Replace
r overwrite character under cursor
R overwrite multiple characters
:s/{term}/{replacement}[/[g]] replace first or all matches in sentence
:{from},{to}s/{term}/{replacement} replace all matches between line numbers
:%s/{term}/{replacement} replace all matches in the buffer
Add
a add after cursor
A add to end of line
o add below line
O add above line
Undo/Redo
u undo last command
CTRL r redo last command
Save
:w save buffer
:w {filename} save buffer to filename
:{from},{to} w {filename} write portion of buffer to filename
Windows
SPC {num} switch to window
SPC w w switch to other window
SPC w j switch to window below
SPC w k switch to window above
SPC w . H move window left
SPC w . L move window right
SPC w . J move window down
SPC w . K move window up
SPC w . [ change window width
SPC w . ] change window width
SPC w . { change window height
SPC w . } change window height
Font
SPC z x + increase font size
SPC z x - decrease font size
SPC z x 0 reset font size
Shell
!{cmd} ENTER execute shell command
SPC ' open shell in new window

Clojure

Evaluate
, e f evaluate form
, e ; evaluate form, result as comment
, e b evaluate buffer
, e v evaluate the sexp
, e n n evaluate namespace
, d v l inspect evaluation results
Structural Editing
SPC k w wrap expression
SPC k s slurp from right
SPC k S slurp from left
SPC k b barf to right
SPC k B barf to left
SPC k r raise
Refactoring
, r f u find usages
Format
, = f format function
, e b format buffer
REPL
, ' start a repl
, s a switch to repl buffer
, m b manage repl connections
, m i show repl connection
Documentation
, h h read function documentation
